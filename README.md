# Data Visualization & Analysis Projects

This repository contains a collection of data analysis, network analysis, and machine learning visualization projects completed during my academic studies. The projects present various end-to-end data science workflows, including data cleaning, exploratory analysis, modeling, and visual communication of insights.

## Projects Overview

### 1. Exploratory Data Analysis (Austin Animal Center)
**Directory:** `01-exploratory-data-analysis`

An in-depth exploratory data analysis (EDA) of animal shelter data from the Austin Animal Center. This project involves processing and analyzing multiple datasets (intakes and outcomes) to identify trends in animal shelter admissions, adoption rates, and other outcomes over time.
- **Topics:** Data wrangling, feature engineering, missing value handling, and statistical visual distributions.

### 2. Network Analysis (Prague Integrated Transport)
**Directory:** `02-network-analysis`

A project focused on applying network theory to public transportation. Using GTFS (General Transit Feed Specification) data from the Prague Integrated Transport (PID) system, this project models transit stops and connections mathematically as a graph. 
- **Topics:** Graph construction, centrality measures, connectivity, and complex network visualizations.

### 3. Machine Learning Visualization (Global Waste Management)
**Directory:** `03-ml-visualization`

This project applies machine learning techniques to real-world country-level data to interpret global macroeconomic and solid waste management indicators. It focuses on using data visualization not just for initial exploration, but for interpreting model paths, understanding feature distributions, and presenting complex machine learning concepts accessibly.
- **Topics:** Predictive modeling/clustering, model evaluation visualizations, and data-driven storytelling.

## Languages & Tools
* **Languages:** Python
* **Environment:** Jupyter Notebook
* **Core Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn

## Getting Started

Each project is contained in its own folder with the respective datasets and a primary Jupyter Notebook (`.ipynb`).

To run the notebooks locally:
1. Clone this repository.
2. Install the necessary data science dependencies. 
3. Navigate to a project folder and launch Jupyter.
